Airbnb Data Analysis Project

Project Overview
This project focuses on analyzing Airbnb Open Data to uncover insights related to pricing, demand, and availability. The analysis was performed using Excel, SQL, and Power BI.

 Objectives
- Analyze pricing trends across different locations  
- Identify demand patterns using reviews data  
- Compare room types and their impact on pricing  
- Understand availability distribution of listings  
- Build an interactive dashboard for visualization  

 Dataset
- Source: Airbnb Open Data (Kaggle)  
- Records: 100K+ listings  
- Features: Price, Room Type, Location, Reviews, Availability, Host details  

Tools & Technologies
- Excel → Data Cleaning  
- SQL → Data Analysis (GROUP BY, ORDER BY, CASE)  
- Power BI → Dashboard & Visualization  

 Key Analysis
- Pricing Analysis (Min, Max, Avg Price)  
- Location-wise Price Comparison  
- Room Type Distribution  
- Demand Analysis using Reviews  
- Availability Categorization (Low, Medium, High)  

 Dashboard Features
- KPIs (Avg Price, Total Listings, Reviews)  
- Price by Location  
- Room Type Distribution  
- Monthly Review Trends  
- Top Hosts by Reviews  
- Availability Status (CASE analysis)  

 Key Insights
- Manhattan & Brooklyn have the highest demand and listings  
- Prices vary significantly based on location  
- Entire homes/apartments are more expensive  
- Few hosts generate most of the reviews  
- Availability patterns indicate supply-demand imbalance  

 Conclusion
This project demonstrates an end-to-end data analytics workflow, from data cleaning to visualization, helping generate actionable business insights.
